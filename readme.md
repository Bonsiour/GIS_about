- ### 实现反距离权重插值算法

  - #### 输入参数

    - ##### 离散点点集

    | x    | y    | z    |
    | ---- | ---- | ---- |
    | 0    | 0    | 0    |
    | 100  | 100  | 200  |
    | 300  | 300  | 500  |
    | 500  | 500  | 200  |
    | 700  | 700  | 500  |
    | 900  | 900  | 200  |
    | 1000 | 1000 | 0    |

    - ##### 像元大小。像元大小决定网格规模（行列数），本示例默认值为10，插值网格规模为100×100。

    - ##### 实现原理：遍历网格，依次计算估计值。详见IDW算法。

    - ##### 实现结果示例：生成矩阵，保存至二维数组，并进行可视化。

      https://github.com/kanvina/GIS_about/raw/master/img/pic_IDW.png
